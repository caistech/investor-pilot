# CROSS-PROJECT NOTES (apply to both)

## What advisor follow-ups need

When an advisor replies positively to a cold Sprint 1 message ("yes, tell me more about Branscombe / Seafields"), the standard packet they receive is:

1. **One-page project teaser** (next deliverable — derived from this term sheet, shorter, marketing-friendly, compliance-reviewed)
2. **This term sheet** (the completed version of file 10)
3. **F2K capability statement** (separate deliverable D-F4 in file 09 — Uwe to produce)
4. **Subscription / EOI form** (digital, captures wholesale-investor declaration + KYC trigger)

Until items 1-4 exist in completed form, Sprint 1 outreach cannot land follow-ups credibly.

## What stays out of the messaging

The Sprint 1 outreach itself **does NOT mention specifics** — no interest rate, no $125K minimum, no return target. Those are revealed only after:
- Connection acceptance (LinkedIn) or first email reply
- Advisor confirms their client is a wholesale investor
- Operator (Dennis or Uwe) confirms the specific advisor + client are appropriate fit
- Then the term sheet is shared via secure channel

This is to:
- Stay clean of anti-hawking concerns (s992A)
- Avoid cold-message compliance issues (no specific terms in unsolicited contact)
- Reserve specifics for verified wholesale audience

## Open question for Uwe + counsel

**Is the advisor-channel investor offer at Branscombe an "open" offer, or is the $500K M0 investor cash deposit the only investor tranche the SPV is taking?** This is the single highest-leverage clarification — it determines whether InvestorPilot's outreach has capacity to fill, or whether the project is already fully equity-funded and the advisor channel should focus elsewhere (Seafields, future projects).

If the $500K is the full investor tranche for Branscombe and it's already committed, **the advisor channel's near-term play is**:
- Build relationships with advisors NOW so that future projects (Homes TAS tender, additional Seafields tranches if structure allows, etc.) have a warm advisor base ready to subscribe
- Use Branscombe + Seafields as evidence of capability, not as live deals

This is a very different Phase 1 posture than "raise $X across these two projects starting today." Worth deciding explicitly.
